# Musique de l'Invitation

## Fichier Audio

Placez votre fichier audio ici sous le nom `invitation-piano.mp3`

### Recommandations

**Style Musical :** Néo-classique / Chamber Pop / Piano solo
- Piano Steinway doux
- Quatuor à cordes aérien
- Ambiance intime et romantique
- Tempo lent (60-80 BPM)

### Format

- **Format :** MP3 ou OGG
- **Durée :** 2-3 minutes (boucle automatique)
- **Volume :** Normalisé à -14 LUFS

Si aucun fichier n'est présent, la musique sera chargée depuis une URL externe libre de droits.
